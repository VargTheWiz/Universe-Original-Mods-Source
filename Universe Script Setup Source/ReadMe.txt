SourceFiles - папка с файлами, которые должны будут попасть в корневую папку игры. Как минимум, там должна быть папка Mods.
Patch - папка, в которой размещается актуальный exe-файл игры.
ModCFG - два конфига со списком подключенных модулей. Первый содержит в себе моды раздела Shu`s Rangers, другой - "чистый" Universe.
Build - папка, куда попадает собранный пакет.

Universe.iss - готовый для билда скрипт.
Также необходим Inno Setup 5.6.1 Unicode версия. - https://files.jrsoftware.org/is/5/innosetup-5.6.1-unicode.exe
