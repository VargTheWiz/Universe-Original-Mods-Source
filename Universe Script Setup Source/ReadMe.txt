SourceFiles - папка куда где лежат файлы которые должны будут попасть в корневую папку игры. То есть Mods как минимум там должен лежать.
Patch - папка где размещается свежий exe файл игры
ModCFG - два конфига со списком подключенных модулей, один с Shu`s Rangers модами, второй чистый Universe
Build - папка куда попадает собранный пакет

Universe.iss - сам скрипт где всё готово для билда.
Необходим Inno Setup 5.6.1 Unicode версия. - https://files.jrsoftware.org/is/5/innosetup-5.6.1-unicode.exe
